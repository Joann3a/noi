### 1. 夏令营旗手
2014 年江苏省《信息与未来》小学夏令营在金陵中学河西分校进行，组委会决定从河西分校的学生中，推选一位小旗手，并输出其编号。河西分校的每一位同学都有一个汉语拼音的名字，名字中的字符全部为大写的英文字
母，且无空格，例如：<br>
名字：王小明，拼音： WANGXIAOMING<br>
每个大写字母都有一个对应的 ASCII 码，如下表：<br>
字符|ASCII<br>
 A  |65<br>
 B  |66<br>
名字中各字符的 ASCII 码的和即为该学生的编号。例如：名字： ‘ABCD’，编号： 266 （即： 65+66+67+68=266）。你的任务是根据给出的学生的名字，输出对应的编号。<br>
【输入】 一个长度不超过 20 的大写字母的字符串。<br>
【输出】 对应的学生编号。<br>

#### 测试数据：
【输入样例】‘ABCD’<br>
【输出样例】266<br>
